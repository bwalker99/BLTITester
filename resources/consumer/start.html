Basic LTI Tester

This utility provides a method of testing both LTI Providers and Consumers. For test purposed they can both be on the same server. For example, to run these test on your own local version of Tomcat:
ToolProvider: http://localhost:8080/bltiprovider/login -- the secret is hardcoded to 'mysecret123'
This is the url provided as the default for the launch paramaters below
ToolConsumer: http://localhost:8080/blticonsumer/start.html (This page)

To test this consumer to this provider on the same host, follow the example below (change host names if need be).

Other Test Tool Consumers: http://ltiapps.net/test/tc.php
(go to this site and enter http://localhost:8080/bltiprovider/login as the Launch URL and mysecret123 as the Shared secret, click 'Save Data', then click 'launch TP')
