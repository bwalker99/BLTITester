Basic LTI Tester

This utility provides a method of testing both LTI Providers and Consumers. For test purposed they can both be on the same server.

For example, when installed on your own local version of Tomcat:

ToolProvider: http://localhost:8080/bltiprovider/login -- the secret is hardcoded to 'mysecret123'. There is no interface. The return will be authentication status and a list of supplied parameters
ToolConsumer: http://localhost:8080/blticonsumer/start.html (This page) - To test Tool Provider on localhost, click either one of the submit buttons.

To test a different Tool Consumers to this Tool Provider:
- go to http://ltiapps.net/test/tc.php and enter http://localhost:8080/bltiprovider/login as the Launch URL and mysecret123 as the Shared secret, click 'Save Data', then click 'launch TP'

To test a different Tool Provider with this Tool Consumer:
- Change the 'Launchsite' field to http://ltiapps.net/test/tp.php, and the 'Secret' field to secret and click Submit
